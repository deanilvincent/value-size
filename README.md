# Overview
NPM Package for getting the size of an array, object or string.

# Installation

`npm i value-size --save`

# Setup & Basic Usage

```
const size = require('value-size')

size([1, 2, 3, 4, 5]); // 5
size('size'); // 4
size({ one: 1, two: 2, three: 3 }); // 3
```
